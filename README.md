Projet - Système de vote

Mode d'emploi - processus:
1.registerVoter
2.registerPropositions
Enregistre les propositions
3.addProposal
Enregistre les propositions de  vote
4.setProposalsRegistrationEnded
5.setVotingSessionStarted
6.addMyVote 
Voter par le numéro de proposition de son choix 
7.setVotingSessionEnded
8.countVotes
9.getWinner
Numéro de la proposition ou propositions ayant eu le plus de votes.

Note: pour exécuter les actions(fonctions) il faut avoir les droits. Soit être le propriétaire (owner) du contrat soit avoir été enregistré dans la liste de votants. 

Le projet a été directement implémenté en mode VotingPlus, les deux fichiers livrés contiennent la même implémentation.
